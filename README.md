# Air Niger — Frontend React

## Description

Application de gestion de compagnie aérienne.
Frontend développé avec ReactJS.

## Technologies utilisées

- ReactJS
- Axios
- React Router DOM

## Prérequis

- Node.js >= 16
- npm

## Installation

### 1. Cloner le projet

git clone https://github.com/Yaou5/compagnie-aerienne-front.git

### 2. Installer les dépendances

npm install

### 3. Lancer l'application

npm start

### 4. Ouvrir dans le navigateur

http://localhost:3000

## Fonctionnalités

- Authentification multi-rôles (Admin/Utilisateur)
- Recherche de vols par origine, destination et date
- Réservation aller simple et aller retour
- Mes réservations
- Annulation de réservation
- Tableau de bord Admin
